IIT Resume Builder

Write in ALL CAPS as it should appear on the resume.

Do not round CGPA. Use the exact value from your official grade card.

Add, rename, reorder, or remove sections. Each section can hold entries with bullets, plain bullet lists, skill rows, or inline text.

Header
  • Name and Roll Number must appear on the resume.
  • LinkedIn, GitHub, Google Scholar links are allowed.
  • No photos. Only the IIT logo is permitted.
Education
  • Do not round CGPA. Use exact value from grade card.
  • Mention CGPA as it appears on grade card.
  • Do not report intermediate % (11th+12th) as 12th %.
7.99 ≠ 8.0  ·  8.56 ≠ 8.6  ·  Use the exact figure.
Projects
  • Course projects must mention the course name/code.
  • Label course projects as [Course Project] in resume.
  • Personal projects can include a link to code or report.
Internships
  • Proof: Completion Certificate or Offer + Ack. mail.
  • Research internships: letter from professor with title & dates.
  • Conference participation: participation certificate required.
Ongoing courses
  • Mark ongoing courses with * in coursework section.
  • Courses registered but not started: mark with **.
Skills & Hobbies
  • No proof required for skills or hobbies.
ATS Best Practices
  • Use standard section names: Education, Experience, Projects, Skills.
  • Use real text — not images — for any meaningful content.
  • Date formats must be consistent (e.g. Mon 'YY or YYYY).
